## Overview

This project is a static frontend clone of the Amazon India homepage. It was built as a frontend development exercise to practice and demonstrate proficiency in core web technologies — semantic HTML5, responsive CSS3 layouts, and vanilla JavaScript DOM interactions.

The project replicates key UI sections of the Amazon homepage including the navigation bar, hero banner, product category grid, and footer.

---

## Features

- 🎨 **Pixel-accurate UI** — closely mirrors the Amazon India homepage layout
- 📐 **Responsive Design** — adapts across desktop and tablet screen sizes
- 🧭 **Navigation Bar** — includes logo, search bar, location selector, cart icon, and sign-in section
- 🖼️ **Hero Banner** — full-width promotional banner section
- 📦 **Product Category Grid** — multi-section product category cards layout
- 📜 **Footer** — multi-column footer matching Amazon's style
- ⚡ **Zero Dependencies** — built entirely with vanilla HTML, CSS, and JS
- 🚀 **Firebase Hosted** — live and accessible via Firebase Hosting

## Getting Started

No build tools or package installation required — this is a pure static project.

```bash
# 1. Clone the repository
git clone https://github.com/rohandeo2609/AmazonClone.git

# 2. Navigate into the project directory
cd AmazonClone

# 3. Open in browser
open public/index.html
```

Or simply open `public/index.html` directly in any modern web browser.

---

## Deployment

This project is deployed using **Firebase Hosting**.

To redeploy:

```bash
# Install Firebase CLI (if not already installed)
npm install -g firebase-tools

# Login to Firebase
firebase login

# Deploy to Firebase Hosting
firebase deploy
```
